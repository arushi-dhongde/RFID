RFID BASED DOOR LOCK SYSTEM

This report is based on the smart door locking technology to develop a Smart door lock system using RFID sensor, numpad and Arduino Uno that allows convenience and security for the users in the field of door lock system. In this project, RFID based smart door lock system is proposed. This system is designed to simulate an electronic solenoid lock, which is controlled through a RFID tag. Moreover, this door lock system is the combination of different technologies in which RFID tag work as a task handler, RFID reader as command agent, Arduino UNO as controlling and data processing unit and solenoid lock as the output unit .


Components Used


1. Arduino
2. I2C Converter
3. LCD display
4. Solenoid lock
5.RFID kit
6.Breadboard








Conclusion

RFID door lock systems offer enhanced security, convenience, and control over traditional lock systems.
They leverage RFID technology, communication protocols, and secure authentication mechanisms to provide efficient access control. While there are still areas for improvement, such as addressing security vulnerabilities and optimizing power management, the existing research and development efforts have paved the way for the successful implementation of RFID door lock systems.
